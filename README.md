# Bot de Envio Automático de Mensagens WhatsApp

Automação em Python que envia mensagens pelo WhatsApp automaticamente.

## O que faz:
- Abre o WhatsApp Web automaticamente
- Envia mensagens para qualquer número no horário definido
- Ideal para empresas que precisam avisar clientes em massa

## Tecnologias:
- Python
- pywhatkit

## Como usar:
1. Instale as dependências: `pip install pywhatkit`
2. Edite o número e a mensagem no arquivo `enviar_mensagem.py`
3. Execute: `python enviar_mensagem.py`
